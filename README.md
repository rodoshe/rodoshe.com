# 🌿 Subat Rodoshe — Portfolio

> A Minecraft-themed personal portfolio website built with HTML, CSS, and JavaScript.

🌐 **Live site:** [rodoshe.com](https://rodoshe.com)

---

## ✨ Features

- 🎮 Minecraft-inspired design — pixel art, item frames, wood/stone/dirt block sections
- 🌙 Dark mode toggle with twinkling stars
- 🖼️ Photo gallery with auto-cycling transitions and dot navigation
- ⌨️ Typewriter animation cycling through roles
- 💥 Particle burst effect on click
- 🐔 Walking chicken across the bottom of the screen
- 🌸 Pixel tulip that appears on the chest hover in the resume section
- ☁️ Animated drifting clouds
- 📱 Mobile responsive

## 🏃 Running locally

just open `index.html` in your browser — no setup needed!

```bash
git clone https://github.com/rodoshe/rodoshe.com.git
cd rodoshe.com
open index.html
```
